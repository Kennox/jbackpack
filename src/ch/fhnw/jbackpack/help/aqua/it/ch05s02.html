5.2. Directory temporanea

La directory temporanea viene usata per archiviare i file creati solo temporaneamente durante il backup e il ripristino. Generalmente occorre disporre di uno spazio libero pari a quello che occupa il file più grande del backup o ripristino.

Generalmente è consigliabile usare la directory di sistema predefinita. Se la directory di sistema è troppo piccola, è possibile configurare una directory differente. L'utilizzo di archiviazione dell'attuale direcotory temporanea configurata è visualizzato al di sotto del percorso.
