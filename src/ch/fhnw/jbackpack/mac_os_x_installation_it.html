Per installare rdiff-backup bisogna eseguire i seguenti passi:
  1. Installare Apple Xcode programming suite, disponibile
    in una installazione separata, dal vostro OS X CD o DVD,
    o preferibilmente l’ultima versione dalla pagina degli sviluppatori Apple:
    http://developer.apple.com/tools/xcode/index.html.

  2. Installare MacPorts da http://www.macports.org.

  3. Aprire Applicazioni → Utilità → Terminale e digitare i seguenti
    comandi (potete copiarli riga dopo riga da qui in basso e
    inserirli nel Terminale, confermare premendo il tasto
    Enter e attendere che ogni singolo comando sia eseguito):
    sudo port selfupdate
    sudo port install rdiff-backup
    sudo bash
    echo "setenv PATH $PATH" >> /etc/launchd.conf

  4. Riavviare il sistema.
